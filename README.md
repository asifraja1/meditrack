## Overview
MediTrack is a Java-based console application for managing doctors, patients, appointments, and billing.
The project demonstrates core Java, OOP principles, collections, generics, and design patterns.

## Features
- Doctor & Patient CRUD
- Appointment booking and status management
- Billing with tax calculation
- Search functionality
- Menu-driven console UI
- Manual testing via TestRunner

## Tech Stack
- Java
- Gradle
- Collections (ArrayList, HashMap)
- OOP & Design Patterns

## How to Run
1. Clone the repository
2. Open in IntelliJ
3. Run `Main.java`

## Testing
Manual tests are implemented in `TestRunner.java`.
